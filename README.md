# Digital-Piano-using-a-Digital-to-Analog-Converter

This course project used 3 buttons, 4 resistors set up as a 1/2/4/8 resistance ratio, and a headphone jack. The board used was an STM32F0DISCOVERY.

Note that this repository only contains the main files.
